📖 Introduction

RCC-PMO User Guide

A complete reference for the Al Rugaib Construction Co. Project Management Office system — covering all modules, workflows, and features available in the current version.

RCC-PMO is a real-time, multi-user project management application serving the PMO, Compliance & Integration, and Facilities Management divisions. All data is synced instantly across all connected users via Firebase, so you always see the latest information regardless of which device or browser you use.

📊
Live Dashboard
Real-time stats, overdue alerts, charts, and a quick-access button to your Daily Task Log.
🤖
AI-Powered Import
Drop any file — PDF, Word, image, or Excel — and AI extracts tasks automatically.
📝
Daily Task Log
Excel-style daily timesheet that pre-populates from your assigned tasks. Saves directly to task records.
🔄
Real-Time Sync
Firebase-powered sync keeps all 37 users in sync instantly — no refresh needed.
👥
Live Presence
Green dots show who's online right now. The top bar shows total connected users.
📄
PDF & Excel Export
Generate formatted PDF reports or export task logs to Excel — all done in the browser.
💡
The app runs entirely in your browser and is accessible at sarchamrcc-ops.github.io/RCC-PMO. Bookmark it or use the 📌 Create Desktop Shortcut button on the Dashboard to pin it to your taskbar.
🔐 Access

Login & Access

The app uses a username + password login. Your credentials were provided by the system administrator. On first login you will be prompted to change your password.

🔑
How to Log In
  1. Open the app URL in your browser and you'll see the login screen.
  2. Enter your username (lowercase, e.g. amadani) and your password.
  3. Press Enter or click Login.
  4. If this is your first login, you'll be prompted to set a new password immediately.
  5. Your session is saved in the browser — you stay logged in across page refreshes until you log out.
🔒
Changing Your Password

Go to your name/avatar → Profile in the top-right of the app. Under Change Password, enter your current password and your new password (minimum 6 characters). Click Update Password.

💡
If you forget your password, contact the system administrator (Ahmad Madani) to have it reset from the Admin panel.
👤 Permissions

Roles & Permissions

Every user account has two independent access controls: a System Role (what you can do) and a User Type (your access level).

User Types

TypeWhat they can do
USER Standard access: create & edit tasks, log time, use Daily Log, Bulk Import. Visibility is scoped to their division and reporting hierarchy.
VIEWER Read-only: can view the Dashboard and task lists but cannot create, edit, or delete tasks or log time.

Divisions

🏗️ PMO 📋 Compliance & Integration 🔧 Facilities Management 👔 Executive

Your division determines which other users' tasks you can see in the People, Projects, and Kanban views. Admins see all divisions.

📊 Dashboard

Dashboard

The Dashboard is the first screen after login. It gives you a live snapshot of everything happening in your scope.

📊
What's on the Dashboard
  • Stat cards: Total Tasks, Active (In Progress), Overdue, Done, and Critical — filtered to your visible scope.
  • Today's Hours + My Avg Progress: How many hours you've logged today and the average progress % across your tasks.
  • Tasks by Status: A visual bar chart showing the breakdown across Backlog, In Progress, On Hold, Review, and Done.
  • Tasks by Type: Breakdown by task type (Design, Meeting, RFI, etc.).
  • Hours by Project: Bar chart showing effort distribution across active projects for the selected period.
  • Overdue Tasks: A list of all overdue tasks within your scope, with direct links to open them.
  • Period selector: Filter all charts by Today, This Week, This Month, This Quarter, This Year, or All Time.
🛠️
Dashboard Actions
  • 📝 My Daily Log — Opens your Daily Task Log for today directly.
  • 📌 Create Desktop Shortcut — Installs the app as a desktop shortcut (works on Windows, macOS, and Android).
  • 📄 Export PDF Report — Generates a multi-page A4 landscape PDF report for the selected period, including stats, charts, and a full task table.
📋 Tasks

My Tasks

The My Tasks view lists all tasks visible to you. You can filter, search, and group them to find exactly what you need.

🔍
Filters & Search
  • Search bar: Searches task title and description in real-time as you type.
  • Status filter: Filter by Backlog In Progress On Hold Review Done.
  • Criticality filter: Critical, High, Normal, Low.
  • Project filter: Filter to a specific project.
  • Type filter: Filter by task type (Design, Meeting, RFI, Submittal, etc.).
  • Assignee filter: Managers and admins can filter to any visible user.
  • Group by: Group the list by Status, Project, or Assignee.

🗂️ Task Detail

Task Detail Modal

Click any task title to open the full task modal. It has a two-panel layout: Edit fields on the left and Updates / Time Log on the right.

✏️

Click 💾 Save Changes to save. Required fields (Project, Type, Assigned To) are highlighted in red if missing.

💬
Right Panel — Updates Tab

The Updates tab shows a chronological thread of all comments and system events (status changes, transfers, time logs). To add an update:

  1. Type your comment in the text area.
  2. Optionally select a new status from the dropdown to change the task status at the same time.
  3. Click Post Update.

System events (status changes, time logged) are automatically added as grey system entries.

⏱️
Right Panel — Time Log Tab

Log hours worked on a task:

  1. Click the Time Log tab on the right panel.
  2. Enter hours (e.g. 2.5), select the date, and optionally add a note.
  3. Click Log Time. The entry is added to the task history and reflected in all reports.

All time logs are synced in real time to Firebase and visible in the Daily Log and People views.

🔀
Transfer & Delete
  • Transfer Task: Available in the task detail. Pass the current holder to another team member. A system update is logged automatically and the new holder receives an email notification.
  • Delete Task: Available to the task creator or the assigned person. Requires confirmation. Deletion cannot be undone.
✚ Create

New Task

Click ✚ New Task in the navigation bar to open the task creation form.

✚
Creating a Task
  1. Fill in the Title — be clear and specific (e.g. "Review MEP drawings for Block C").
  2. Select the Project from the dropdown.
  3. Choose the Type (Design, Meeting, RFI, etc.) and Criticality.
  4. Set the Status (default: Backlog) and a Due Date.
  5. Assign the task using Assigned To and set Assigned By (requester).
  6. Optionally add a Description and Client name.
  7. Click Create Task. The assignee receives an email notification automatically.
🤖
AI Auto-Fill

Paste any text description of the task into the title or description fields — the AI will suggest the best type, criticality, project, and due date. The AI works with English and Arabic text.

Choose your AI provider (Gemini free / OpenAI / Anthropic) and API key from the AI provider dropdown on the form.

📝 Daily Log

Daily Task Log

The Daily Task Log is your personal timesheet — an Excel-style table for recording how many hours you spent on each task each day. It pre-populates from your assigned tasks so you don't have to type anything from scratch.

📝
Using the Daily Log
  1. Open via Navigation → Daily Log or click 📝 My Daily Log on the Dashboard.
  2. The log defaults to today's date. Use ← Prev Day / Today / Next Day → to navigate between days.
  3. Each row represents one task. Select a task from the dropdown in the first column — it will pre-fill the project name, code, and description.
  4. Enter the hours worked (numbers only, e.g. 3 or 1.5).
  5. Fill in the Progress % if it has changed.
  6. Add a Note (optional).
  7. Click 💾 Save Log. This writes time log entries and progress updates directly to the task records.
📊
Export to Excel

Click 📊 Export Excel to download a multi-sheet Excel workbook. Each sheet represents one calendar day and lists all tasks you logged time on, formatted as the standard RCC daily log sheet.

👁️
Manager View

Managers see an employee selector dropdown at the top and can view any subordinate's log (read-only). Only the employee themselves can save or edit entries.

The People view also has a 📝 Log shortcut button next to each employee's name to jump directly to their log.

⚠️
Saving a daily log entry updates the task's time log and progress in real time. All other users will see the updated hours immediately in their reports and dashboards.
🗂️ Kanban

Kanban Board

The Kanban view shows all visible tasks organised into columns by status. Use it to get a visual overview of workflow bottlenecks and move tasks between stages.

🗂️
Global Kanban
  • Five columns: Backlog In Progress On Hold Review Done.
  • Filter by Project, Status, Assignee, and Criticality using the dropdowns above the board.
  • Drag a card from one column to another to instantly update its status.
  • Click any card to open the full task detail modal.
  • Critical tasks are shown with a red left border.
🏗️
Project Kanban & By-Employee View

Inside a project's detail page, the Kanban and By Employee tabs show tasks for that project only:

  • Kanban tab: Same column layout but scoped to the project.
  • By Employee tab: Each column represents one team member. Drag cards between columns to re-assign tasks within the project.
📅 Calendar

Task Calendar

The Task Calendar displays tasks on a monthly calendar based on their due date.

  • Navigate months with ← Previous / Next → buttons.
  • Filter by Project, Status, Assignee, and Criticality.
  • Click a task badge on the calendar to open its detail modal.
  • Overdue tasks appear with a red accent. Tasks due today are highlighted.

👥
Workload Table

Clicking an employee opens their personal page, which includes:

  • Stats: Tasks Assigned, Projects, Hours Logged, Average Progress.

🏗️ Projects

Projects

The Projects view lists all active projects with task counts, total hours, and focus percentage.

🏗️
Project List
  • Filter by Division to see only relevant projects.

Project Detail

📥 Import

Bulk Import

The Bulk Import module lets you add many tasks at once from a document, spreadsheet, or typed list — with AI-assisted extraction.

🤖
Tab 1: AI File Import
  1. Click 📥 Bulk Import in the nav bar.
  2. Select your AI provider and paste your API key (Gemini is free at aistudio.google.com).
  3. Drag and drop — or click to browse — any file: PDF, Word (.docx), image, Excel, CSV, or plain text.
  4. The AI reads the file and extracts tasks, filling in title, type, criticality, project, assignee, and due date where it can infer them.
  5. Review the preview table. Edit any field that needs correction.
  6. Set the Assigned To employee and click Import.
💡
The AI reads Arabic and English text. For best results, use files that list tasks clearly with action verbs.
📋
Tab 2: Paste from Excel

Copy a range of cells from Excel, paste them directly into the text area. The system expects columns: Title | Project | Type | Assigned To | Criticality | Due Date. Rows are parsed and shown in a preview table for review before import.

✏️
Tab 3: Quick List

Manually type a quick list of tasks row by row using the form table. Click + Add Row to add more tasks, then Preview Import → to review before confirming.

🔄 Sync

Real-Time Sync & Presence

🔄
Firebase Real-Time Sync
  • All data (tasks, projects, users) is stored in Firebase Realtime Database and synced across all connected browsers automatically — no refresh needed.
  • The connection status is shown in the top-right corner of the app.
  • If you go offline, changes are stored locally and synced automatically when you reconnect.
  • Firebase is the source of truth — when you first open the app, it loads data from Firebase (overwriting any older local data).
👥
Online Presence
  • The 👥 badge in the top bar shows how many users are currently connected.
  • Green dots on user avatars in the People view indicate users who are online right now.
  • Presence updates instantly when users log in or close the app.
❓ FAQ

Frequently Asked Questions

I logged in but the data looks outdated. What do I do?

The app loads data from Firebase automatically. If it looks stale, try refreshing the page (Ctrl+R). If the problem persists, contact the system administrator.

I can't see tasks from other divisions. Is that normal?

Yes. By default, you see tasks assigned to you and your division's team. Admins and the COO see all divisions. Contact your admin if you need cross-division visibility.

How do I save my Daily Log?

Fill in the hours and progress, then click the 💾 Save Log button at the top-right of the Daily Log view. Data is not saved automatically — you must click Save.

The AI import isn't working. What's wrong?

Check that your API key is correct for the selected provider. Gemini is free at aistudio.google.com. OpenAI and Anthropic require paid accounts. Check your internet connection if the key looks correct.

I forgot my password. What do I do?

Contact the system administrator (Ahmad Madani) to reset it from the Admin panel.

Can I use this on my phone?

Yes. The app is responsive and works on mobile browsers. Use the 📌 Create Desktop Shortcut button on the Dashboard to add it to your home screen for quick access.

Will I lose data if I close the browser?

No. All saved data is synced to Firebase. Your session is also remembered, so you won't need to log in again on the same device/browser.
