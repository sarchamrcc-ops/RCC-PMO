What's Inside

Everything in one place

RCC-PMO is your team's task management hub — built for construction project oversight across PMO, Compliance & Integration, and Facilities Management.

📊
Live Dashboard
Real-time stats showing active tasks, hours logged, overdue items, and project summaries at a glance.
🗂️
Task Management
Create, assign, and track tasks through a Kanban board with full status workflow from Backlog to Done.
⏱️
Time Logging
Log hours per task, import from Excel timesheets, and export full per-employee task logs.
📋
Task Library
40+ pre-built task templates across 7 categories. Start a task instantly without typing from scratch.
📄
PDF Reports
Generate professional progress reports with charts, project breakdowns, and per-user summaries.
🔁
Real-time Sync
Firebase-powered sync means all users see updates instantly — no refresh needed, no data conflicts.
🏢
Division Access
PMO, Compliance & Integration, and FM teams each see their own tasks and projects. Admins see all.
🔔
Notifications
Get alerted when tasks are assigned to you, due dates approach, or when colleagues update your tasks.
📤
Excel Import
Drop in daily log sheets (named 1–31) and the system auto-creates tasks with hours — zero rekeying.

Time Log & Excel Import

The Time Log section shows every hour recorded against tasks. For bulk entry, import your monthly Excel log sheet and the system does the rest automatically.

Name your Excel sheets 1, 2, 3…31 (one per calendar day) for auto-detection
Use Batch Import to drop multiple employee files at once
Filter by period (this week / this month / custom) to see relevant hours
Click Export Excel on any person's row to download their full task log

Quick Reference

Handy Shortcuts & Tips

Get the most out of RCC-PMO with these quick reference cards.

🗂️ Task Status Guide
  • Backlog Task created but not yet started
  • In Progress Actively being worked on
  • On Hold Paused — waiting on something
  • Review Submitted for approval / check
  • Done Completed and closed
⚡ Navigation
  • Dashboard Overview of all stats & charts
  • My Tasks Your personal Kanban board
  • All Tasks Full org task list (managers)
  • Time Log ⏱ Hours tracking & Excel import
  • Reports 📄 PDF generation & data export
  • Task Library 📚 40+ ready-made templates
  • Admin ⚙️ User management (admins only)
📥 Excel Import — How To
  • Step 1 Prepare your Excel file with sheets named 1, 2 … 31
  • Step 2 Go to Time Log → click Import Excel
  • Step 3 Select the file — system auto-reads all daily sheets
  • Step 4 Review the import preview and confirm
  • Batch Use Batch Import for multiple employees at once
🆘 Common Questions
  • Can't log in? Contact Ahmad Madani or System Admin u0
  • Wrong division? Admin can reassign your division anytime
  • Data not updating? Check your internet — app uses live sync
  • Can't see a task? It may belong to another division
  • Password change? Profile icon → Change Password
  • App URL sarchamrcc-ops.github.io/RCC-PMO
